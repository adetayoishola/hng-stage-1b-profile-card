# HNG Internship - Stage 1b: Testable Profile Card

A clean, responsive and accessible Profile Card built for HNG Frontend Stage 1b.

## Features
- Fully responsive design (Mobile, Tablet & Desktop)
- Live updating current time in milliseconds
- Semantic HTML with proper accessibility
- Social links that open in new tab
- Hobbies and Dislikes sections
- All required `data-testid` attributes implemented

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
